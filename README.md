# Forecasting-Workload-Analysis

# This project analyses the distribution of working hours to support forecasting and workload planning.

# The objective was to identify patterns in time allocation and build a simple forecasting approach to estimate future workload demand.

## Tools Used
- Excel
- Data Analysis
- Forecasting Techniques

## Key Analysis
- Distribution of hours across tasks
- Identification of workload patterns
- Forecasting of future workload demand

## Insights
The analysis helped identify peaks in workload distribution and provided a simple forecasting model to support resource planning.
